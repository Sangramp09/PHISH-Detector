# 🛡 PHISH-Detector

AI-powered phishing email detection system built using Flask, Machine Learning, OCR

## Features

- Email text analysis
- Screenshot OCR scanning
- Risk score generation
- Safe / Phishing prediction

## Tech Stack

- Python
- Flask
- Scikit-Learn
- Tesseract OCR
- HTML
- CSS
